## 🚀 How to Run Locally

1. Download and extract the ZIP
2. Open the `Majestique-Landmarks/` folder
3. Double-click `index.html` — opens in any browser
4. **No server or build tools required**

> **Note:** The hero video (`assets/video/hero.mp4`) must stay in the same folder structure. All other assets (images, CSS, JS) are referenced relatively.

---

## 🌐 To Deploy on a Web Server

Upload the entire `Majestique-Landmarks/` folder contents to your hosting root (e.g. `public_html/` on cPanel):

```
public_html/
  index.html
  css/style.css
  js/main.js
  assets/images/...
  assets/video/...
```
